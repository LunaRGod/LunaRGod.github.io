# 函数

py 中使用 def 关键字来定义函数

参数

括号中如果调用并没有写入参水那么使用默认的值 China 如果传入了那么就使用默认的值

传递到函数的参数可以是任何数据类型 (字符串 数字 列表 字典)

函数内部需要返回值那么使用 return 接收，调用函数时可以 print 打印亦或者使用变量进行保存

关键字参数

可以使用特定的 key=value 语法传参，赋予姓名调用时也赋予姓名，避免了因为顺序问题而发生混淆

任意参数

不清楚给予函数传递的是几个参数，那么在函数定义的参数名称前添加 *

# Lambda 匿名函数

Lambda 函数是一种小的匿名函数，可以接受任意数量的参数，但只能有一个表达式，通常用于在不需要定义完整函数的情况下，快速定义和使用简单的函数。

lambda 可以做为另一个函数的匿名函数来使用

# _init_() 函数

内置函数 _init_() 函数，所有类都有这个内置的函数，它始终在启动类时执行，使用这个函数将赋值给对象，或者在创建对象时需要执行其他的操作

self 是特殊的参数，表示类的实例也就是本身，虽然可以选择其他名称但是通常会使用这个

# 对象方法

对象方法等于类中的方法，属于该类的函数

在类中定义的方法无聊用不用都需要加上 ** self 参数，不在类中定义而是 ​ python 的普通方法则不需要直接和 JS ** 一样定义就可以了

迭代器是一种对象，包含的值可计数数字，意味着可以遍历所有的值，它包含方法 _iter_() _next_()

列表、元组、字典 、集合都是可以迭代的对象，它是可迭代的对象，可以从中获取迭代器 (Iterator) , 所以这些对象都有用于获取迭代器的 iter() 方法

# 创建迭代器

将对象 / 类创建为迭代器，必须为对象实现 _iter_() 和 _next_() 方法，所有的类都有初始化的

_init_() 函数，它允许在创建对象时进行一些初始化，

_iter_() 方法的作用相似，它可以执行操作 (初始化) 但必须始终返回迭代器对象本身

_next_() 方法也允许你执行操作 并且必须返回序列中的下一个项目
